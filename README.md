## How to use it
### 1- Install [conky](https://github.com/brndnmtthws/conky) for linux (all command must be run as sudo, doas or sup)
#### Debian / Debian based:
    apt install conky
#### ArchLinux / Arch based:
    pacman -S conky
#### Fedora:
    dnf install fastfetch

### 2- Download [this](https://github.com/Herra-dev/Conky_config/blob/main/conky/conky.conf) conky config and move it ~/where/you/want

### 3- Lauch conky with this command (replace conky.config with your conky config name)
    conky -c conky.config
